# Lab 7 Report: AutoGen vs CrewAI

## Overview

In this lab, I explored two multi-agent frameworks: AutoGen and CrewAI. Both frameworks allow multiple AI agents to work together, but they organize collaboration in different ways.

AutoGen uses a conversational workflow, where agents communicate with each other in a shared group chat. CrewAI uses a task-based workflow, where each agent has a specific role and completes assigned tasks in a fixed sequence.


## Comparison

The main difference between AutoGen and CrewAI is how the agents collaborate.

AutoGen is conversation-based. Agents interact more freely, so the workflow is flexible and good for brainstorming. However, because the order of conversation can change, it may be less predictable.

CrewAI is task-based. Agents follow assigned tasks in a clearer sequence, so the workflow is easier to understand and control. However, it may feel less flexible than AutoGen when the problem requires open discussion.

## Conclusion

AutoGen is better for flexible, discussion-based collaboration, especially when the problem is open-ended. CrewAI is better for structured workflows where each agent has a specific responsibility and the final output needs to be organized.
